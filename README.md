# 💬 ChatApp Client-Server

Aplikasi chat desktop berbasis Python dengan antarmuka grafis (Tkinter) dan protokol socket TCP.
---

## 🚀 Fitur
- 🧑‍💬 **Login & Nickname**
- 💬 **Chat publik & pesan pribadi (PM)**
- 📁 **Kirim file publik maupun privat**
- 👥 **Daftar user online**
